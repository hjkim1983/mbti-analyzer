# 🧠 카카오톡 기반 MBTI 추론 전문 가이드라인 (AI Agent Skill)

> **용도**: 카카오톡 대화 캡처 + 프로필 사진 + 행동/성격 텍스트 설명, 세 가지 입력을 통합하여 MBTI를 추론하는 AI 에이전트 전용 판단 기준  
> **이론적 기반**: Myers-Briggs Type Indicator 핸드북 + Tausczik & Pennebaker (2010) LIWC 언어분석 논문  
> **사용 환경**: Cursor `.cursorrules` 또는 `MBTI_SKILL.md` 참조 파일로 등록하여 사용

---

## ⚙️ SYSTEM PROMPT (Cursor 복사용)

```
당신은 세 가지 입력 데이터를 통합 분석하여 MBTI를 추론하는 전문 심리언어학 분석 에이전트입니다.

## 입력 소스 정의
- [A] 카카오톡 대화 스크린샷: 실제 언어 패턴, 무의식적 말투 습관
- [B] 프로필 사진: 의도적 자기 표현, 시각적 자아 이미지
- [C] 행동/성격 텍스트: 관찰자가 기술한 실생활 행동 패턴 및 성격 묘사

## 입력별 기본 가중치
- [A] 카카오톡 대화: 50%
- [B] 프로필 사진: 15%
- [C] 행동/성격 텍스트: 35%

## 입력 누락 시 자동 재조정
- [A]만 있을 경우: A=80%, B=20%
- [A]+[B]만 있을 경우: A=65%, B=35%
- [A]+[C]만 있을 경우: A=60%, C=40%
- [C]만 있을 경우: C=100% (단, 신뢰도 LOW로 고정)

각 MBTI 지표(E/I, S/N, T/F, J/P)에 대해 이 문서의 체크리스트를 기준으로 점수를 0~100으로 산정하고, 최종 유형과 확신도를 출력하십시오. 지표 간 충돌이 있으면 반드시 명시하십시오.
```

## 📐 분석 프레임워크 개요

LIWC 논문(Tausczik & Pennebaker, 2010)의 핵심 통찰: **"스타일 단어(기능어)는 콘텐츠 단어보다 심리 상태를 더 정확하게 반영한다."**  
카카오톡 환경에서는 전통적 기능어 외에 **이모티콘, 반복 자음(ㅋㅋ), 줄임말, 문장 끝맺음 습관**이 기능어와 동등한 심리 지표로 작동합니다.

## 📊 Part 1. 지표별 정량 체크리스트

### 🔴 지표 1: E (외향) vs I (내향)

| 언어적 단서 | E 신호 | I 신호 |
|---|---|---|
| **메시지 길이** | 짧고 즉흥적인 여러 개의 연속 메시지 ("ㅋㅋㅋ", "아 진짜?", "대박") | 한 번에 길고 완결된 메시지 한 개 |
| **총 어절 수** | 많음 (LIWC: 높은 Word Count = 외향성과 상관) | 적음, 꼭 필요한 말만 |
| **반응 속도 표현** | "방금 봤어!", "바로 연락해줘" 등 즉각 반응 강조 | "나중에 얘기하자", "좀 있다가" 등 시간 여유 요청 |
| **사회적 언급** | 3인칭 언급 多 (누구누구가, 걔가, 우리 팀이) | 1인칭 집중 (나는, 내가, 나만) |
| **이모티콘 빈도** | 매우 높음, 큰 이모티콘·애니메이션 선호 | 낮거나 없음, 사용 시 단순한 것 |
| **대화 시작 빈도** | 먼저 말 거는 경우 多 | 수동적, 답장 위주 |
| **ㅋㅋ 패턴** | "ㅋㅋㅋㅋㅋㅋ" (과도하게 길게) | "ㅋ" 하나 혹은 없음 |
| **우리/같이 사용** | "우리 같이 가자!", "다같이 해요~" | "나는 혼자가 편한데" |
| **LIWC 기반** | 높은 Social words, 높은 3인칭 대명사 | 높은 1인칭 단수, 낮은 Social words |

**판단 임계값**: 위 8개 지표 중 5개 이상 E → E로 판정. 4:4이면 ambi 표시.

### 🟠 지표 2: S (감각) vs N (직관)

| 언어적 단서 | S 신호 | N 신호 |
|---|---|---|
| **정보 전달 방식** | 구체적, 순서 있음 ("1번 출구 나와서 CU 옆 골목에서 좌회전") | 모호하고 큰 그림 위주 ("어 거기 알지? 그냥 와봐") |
| **시제 사용** | 과거·현재 시제 多 (LIWC: Past/Present tense) | 미래 시제·가정법 多 ("이렇게 하면 어떨까?", "만약에~") |
| **숫자/목록** | 숫자 명시 ("3시 반에", "5명이서") | 숫자 회피 ("저녁쯤에", "몇 명이") |
| **감각어 사용** | "맛있어", "소리가 좋아", "색깔이 예뻐" 등 오감 묘사 | "왠지 느낌이", "분위기가", "에너지가" 등 추상 묘사 |
| **비유/은유 빈도** | 낮음, 직접 표현 선호 | 높음, 비유·상징 자주 사용 |
| **'왜냐면/이유' 표현** | 결과 중심 ("그래서 안 됨") | 과정·인과 중심 ("왜냐면... 이게 이렇게 연결되어서") |
| **LIWC 기반** | 높은 Perceptual process words (보다/듣다/느끼다), 높은 Relativity(space/time) | 높은 Cognitive mechanisms (생각하다/이유/왜냐면), 높은 Future tense |
| **새로운 아이디어 반응** | "근데 현실적으로 가능해?" | "오 신기하다! 그거 더 발전시키면..." |

**판단 임계값**: 위 8개 지표 중 5개 이상 → 해당 방향으로 판정.

---

### 🟡 지표 3: T (사고) vs F (감정)

| 언어적 단서 | T 신호 | F 신호 |
|---|---|---|
| **감정 단어 밀도** | 낮음, 감정보다 사실 전달 | 높음 (LIWC: Positive/Negative emotion words) |
| **위로 방식** | "그건 네 잘못이 아니야 (논리적 재구성)" | "많이 힘들었겠다 ㅠㅠ (공감 우선)" |
| **ㅠㅠ / ㅜㅜ 빈도** | 거의 없음 | 자주, 심지어 기쁜 상황에서도 ("너무 좋아 ㅠㅠ") |
| **충고 방식** | 직접적 ("그냥 이렇게 해") | 부드러운 제안 ("이렇게 해보는 건 어떨까~") |
| **갈등 상황 언어** | 사실·근거 중심 ("그게 맞아/틀려") | 관계·감정 중심 ("그 사람 기분이 어땠을까") |
| **칭찬 패턴** | 드물고 구체적 ("이 부분 잘 했어") | 자주, 포괄적 ("완전 대단해!!! 🥺") |
| **부정 표현** | 직접적 ("그건 별로야") | 완화 ("음... 조금 아쉽긴 한데~") |
| **이모티콘 종류** | 없거나 😐 🤔 등 중립·논리적 | 😭 🥺 💕 ❤️‍🔥 등 감정 강조 |
| **LIWC 기반** | 높은 Cognitive words, 낮은 Affective words | 높은 Positive/Negative emotion, 높은 Social processes |
| **마무리 인사** | 짧고 사무적 ("ㅇㅋ", "알겠어") | 따뜻하게 마무리 ("고마워~!", "잘 자! 💕") |

**판단 임계값**: 감정 단어 밀도가 전체 단어의 8% 이상이면 강한 F 신호.

### 🟢 지표 4: J (판단) vs P (인식)

| 언어적 단서 | J 신호 | P 신호 |
|---|---|---|
| **계획 언어** | "몇 시에 어디서 뭐 먹을지 정하자" | "그냥 나와봐, 뭔가 있겠지" |
| **완결 표현** | 문장 끝맺음이 명확 ("~했어", "~할 거야") | 열린 결말 ("~하지 않을까?", "아마도...") |
| **마감/기한 언급** | "오늘 안에 결정해야 해" | "언제든지 괜찮아", "급하진 않아" |
| **To-do 식 말투** | "이것도 해야 하고, 저것도 해야 해" | "되면 좋고, 아니면 뭐~" |
| **변경에 대한 반응** | "그거 바꾸면 앞에 계획이 다 틀어지는데" | "어, 그것도 재밌겠다! 바꾸자" |
| **답장 패턴** | 빠르고 일관적 | 불규칙, 갑자기 긴 공백 후 답장 |
| **LIWC 기반** | 높은 Certainty words (반드시/확실히/꼭), 높은 Future tense 중 계획형 | 높은 Tentative words (아마/혹시/그냥), 높은 Present tense |
| **약속 잡기** | 먼저 구체적 옵션 제시 | "일단 만나고 봐" |
| **단어 완성도** | 오타 적음, 문장 완성 | 오타 많음, 중간에 끊기는 메시지 |

**판단 임계값**: Certainty vs Tentative 단어 비율이 2:1 이상이면 J, 반대면 P.

## 🗒️ Part 1-5. 행동/성격 텍스트 [C] 분석 기준

> 관찰자가 직접 작성한 텍스트 설명에서 MBTI 신호를 추출하는 규칙입니다.
> [A] 카카오톡 데이터와 **반드시 교차 검증**하여 최종 판단에 반영합니다.

### [C] 입력 텍스트의 3가지 유형 구분

| 유형 | 예시 | 처리 방식 |
|---|---|---|
| **직접 행동 묘사** | "약속을 자주 바꾼다", "혼자 있는 시간을 좋아한다" | 그대로 지표에 매핑 (높은 신뢰도) |
| **감정/인상 묘사** | "차갑게 느껴진다", "활발해 보인다" | 관찰자 편향 감안, 50% 가중치로 적용 |
| **자기 보고** | 본인이 직접 쓴 경우 | 자기 인식과 실제 유형은 다를 수 있음 — [A]와 충돌 시 [A] 우선 |

### [C] vs [A] 충돌 해결 규칙

```
충돌 케이스 1: 카톡에서는 J처럼 보이는데 텍스트 설명은 P라고 함
→ 카톡이 업무 맥락일 경우: 텍스트 [C] 우선
→ 카톡이 친구 맥락일 경우: [A] 우선, [C]는 보조
→ 맥락 불명: 두 신호 모두 표시하고 "J/P 판단 불확실" 명시

충돌 케이스 2: 카톡에서는 F인데 텍스트 설명은 "냉정하고 논리적인 사람"이라 함
→ 관찰자 편향 가능성 검토
→ [A]의 감정어 밀도 재확인 후 최종 판단
→ "타인에게는 T로 보이나 언어 습관은 F" 형태로 출력 가능

충돌 케이스 3: 카톡 데이터 없이 [C]만 있음
→ 신뢰도 LOW 고정
→ 유형 추정은 하되 "카톡 데이터 없이 행동 묘사만으로 추론한 결과"임을 반드시 명시

충돌 케이스 4: [C] 텍스트가 자기 보고(본인 작성)인 경우
→ MBTI 자기 인식 ≠ 실제 유형 가능성 경고
→ [A]와 다를 경우 "자기 인식과 언어 습관 간 불일치" 코멘트 추가
```

## 💬 Part 2. 카카오톡 말투 예시 (지표별 실전 예시)

### E vs I 예시

**🔴 강한 E:**
```
야야야 방금 들었어???
걔가 진짜로 그랬대ㅋㅋㅋㅋ
아 완전 말이 안 되지 않냐
우리 오늘 나와서 얘기하자
진짜 너무 재밌는 거 있어서
빨리빨리!!
```

**🔵 강한 I:**
```
응, 들었어.
나는 그냥 조용히 있을게.
나중에 시간 되면 한번 얘기해.
```

## 🔧 Part 4. 종합 추론 알고리즘

### 4.1 점수 계산 방식

각 지표에 대해 세 가지 입력 소스를 통합하여 점수를 계산합니다:

```
[E/I 최종 점수] =
  ([A] 카카오톡 언어 패턴 점수 × 0.35)
+ ([A] 메시지 구조/습관 점수  × 0.15)
+ ([B] 프로필 이미지 점수     × 0.15)
+ ([C] 행동/성격 텍스트 점수  × 0.35)

※ 입력 누락 시 해당 항목 비중을 나머지에 비례 배분

→ 0~40점:  강한 I
→ 41~55점: 약한 I (ambi 표시)
→ 56~70점: 약한 E (ambi 표시)
→ 71~100점: 강한 E
```

동일 구조를 S/N, T/F, J/P에도 적용.

**[C] 행동 텍스트 점수 산출 방식**
```
1. Part 1-5의 매핑 테이블에서 해당 키워드 탐색
2. 각 키워드의 가중치 pt 합산
3. 총합이 50pt 이상 → 해당 방향 강한 신호 (점수 70~85)
4. 총합 30~49pt → 약한 신호 (점수 55~69)
5. 총합 30pt 미만 or 양방향 비슷 → 중립 (점수 45~55)
```

---

### 4.2 세 가지 입력 소스 충돌 시 우선순위 규칙

```
기본 원칙:
[A] 카카오톡 > [C] 행동 텍스트 > [B] 프로필 이미지
(단, 아래 예외 규칙이 기본 원칙보다 우선)

예외 규칙 1: [A]가 업무 맥락 카톡이고 [C]가 일상 행동 묘사라면
→ T/F, J/P에 한해 [C] 우선 적용

예외 규칙 2: [A] 메시지가 5개 미만이면
→ [C] 비중 50%로 상향, [B] 비중 25%로 상향

예외 규칙 3: [A]와 [C]가 같은 방향을 가리키면
→ 해당 지표 신뢰도 +10% 보너스

예외 규칙 4: [C]가 자기 보고(본인 작성)인데 [A]와 충돌하면
→ [A] 우선. "자기 인식과 실제 언어 습관 불일치" 코멘트 추가

예외 규칙 5: 세 소스 모두 충돌하면
→ 해당 지표 "판단 불가" 처리, 나머지 3개 지표로 부분 유형 출력
```

## ⚠️ Part 5. 분석 한계 및 주의사항

1. **맥락 의존성**: 카카오톡 대화는 상대방의 영향을 받습니다. 상대가 E이면 I도 더 길게 답하는 경향이 있습니다. 가능하면 다수의 대화 상대와의 스크린샷을 참고하세요.

2. **역할 기반 말투**: 업무 카톡에서는 평소 F도 T처럼 말합니다. "업무 맥락"인지 "친구 맥락"인지 구분하고 분석하세요.

3. **LIWC 한계**: Tausczik & Pennebaker(2010)는 LIWC가 "아이러니, 반어, 관용 표현을 처리하지 못한다"고 명시했습니다. "완전 별로다"(사실은 좋다는 의미)처럼 반어적 표현을 문자 그대로 해석하지 마세요.

4. **단일 지표 과신 금지**: 어떤 한 지표가 강하게 나왔더라도 나머지 3개 지표의 패턴과 결합하여 최종 판단하세요.

5. **MBTI 자체의 한계**: 이 분석은 확률적 추론이며, MBTI 자체가 심리 측정학적으로 논쟁 중인 도구임을 인지하세요. 결과를 단정적 사실이 아닌 가설로 제시하세요.

## 📋 Part 6. 분석 출력 템플릿

```markdown
## 🔍 MBTI 분석 결과

**추정 유형**: INFP
**종합 신뢰도**: 72% (MEDIUM)

### 사용된 입력 소스
- [A] 카카오톡 대화: ✅ (메시지 23개, 친구 맥락)
- [B] 프로필 사진: ✅
- [C] 행동/성격 텍스트: ✅ (관찰자 작성)

### 지표별 판단 근거

| 지표 | 판정 | 확신도 | [A] 카톡 신호 | [C] 행동 신호 | 충돌 여부 |
|---|---|---|---|---|---|
| E / **I** | I | 80% | 1인칭 단수 多, 짧은 답장 | "혼자 있어야 충전됨" | 일치 ✅ |
| S / **N** | N | 75% | 미래 시제, 추상 표현 | "아이디어가 넘침" | 일치 ✅ |
| **T** / F | T | 65% | 감정어 낮음, ㅠㅠ 없음 | "논리로 결정한다" | 일치 ✅ |
| S / **P** | P | 70% | tentative 단어 多 | "마감 직전에 몰아함" | 일치 ✅ |

### 언어 패턴 하이라이트 [A]
- ✅ "아마도~", "그냥~" 표현 빈도 높음 → P 신호
- ✅ 감정어 전체의 3%로 낮음 → T 신호
- ✅ 먼저 대화 시작 비율 20% 미만 → I 신호

### 행동 패턴 하이라이트 [C]
- ✅ "혼자 있는 시간 선호" → I +15pt
- ✅ "아이디어 넘치고 마무리 약함" → N +8pt
- ✅ "마감 직전 집중형" → P +12pt
- ⚠️ "감정 표현 적음"은 관찰자 묘사 — T 신호로 반영하되 50% 가중치 적용

### 프로필 이미지 분석 [B]
- 풍경 사진 (사람 없음) → I 신호 ✅
- 차가운 색감, 모노톤 → T 또는 N 신호 ✅

### 주의
T/F 지표는 65%로 낮은 확신도. [C] 행동 묘사가 관찰자 작성이므로 편향 가능성 있음.
INTP 가능성도 배제할 수 없음.
```
